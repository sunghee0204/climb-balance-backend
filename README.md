### 🌸 Overview

| HTTP METHOD |       End Point       |    Description     |
| :---------: | :-------------------: | :----------------: |
|    POST     |     /account/user     |      회원가입      |
|    POST     |    /account/login     |       로그인       |
|     GET     |      /post/feed       |     피드 조회      |
|     GET     |    /account/mypage    |  마이페이지 조회   |
|     GET     | /post/detail/:post_id | 게시글 상세 페이지 |
